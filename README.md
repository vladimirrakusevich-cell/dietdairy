# Diet Water Reminder Bot
Telegram-бот для напоминаний о воде и окне питания. Node.js + Telegraf.

## Переменные окружения
- BOT_TOKEN — токен бота из @BotFather  
- TZ — таймзона, напр. Europe/Moscow  
- DEFAULT_WATER_GOAL_ML — цель воды в мл (по умолчанию 2700)

## Запуск локально
npm install
BOT_TOKEN=xxx TZ=Europe/Moscow npm start
